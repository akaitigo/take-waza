# take-waza

竹細工の編み方パターンをノードグラフとしてモデル化し、3Dプレビューで完成形を確認しながら設計できるCADライクなツール。

## 特徴

- 伝統的な編み方パターン（六つ目編み、網代編み、ござ目編み等）のライブラリ内蔵
- 竹ひごパラメータ（幅・厚さ・本数）のインタラクティブ調整
- Three.js による3Dリアルタイムプレビュー
- 編み手順のステップバイステップ図解生成
- グラフ理論による編みパターンモデル化でパラメトリック変形が可能

## 技術スタック

- **フロントエンド**: TypeScript / React + Three.js
- **計算エンジン**: Rust / WASM（wasm-pack）
- **ビルド**: Vite
- **テスト**: Vitest + Playwright
- **リント**: Biome + oxlint

## セットアップ

```bash
# 依存インストール
npm install

# 開発サーバー起動
npm run dev

# テスト実行
make check
```

## 注意事項

> **本番利用前に必要**: ユーザー認証、データ永続化（API層）、セキュリティヘッダーの設定は MVP スコープ外です。本番環境での利用前にこれらの実装が必要です。
